# Building Micrograd

A small educational project where I reimplement a minimal **automatic
differentiation engine** and a tiny **neural network library** from
scratch in Python.

Inspired by Andrej Karpathy's *micrograd*, this repository focuses on
understanding the core mechanics behind:

-   computational graphs
-   automatic differentiation
-   backpropagation
-   simple neural networks

The goal is to better understand how modern deep learning frameworks
compute gradients under the hood.
